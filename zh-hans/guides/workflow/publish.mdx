---
title: 应用发布
---

调试完成之后点击右上角的「发布」可以将该工作流保存并快速发布成为不同类型的应用。

![](https://assets-docs.dify.ai/2025/03/6cd7d2105cb5a9e4f25601efbda4ffb0.png)

对话型应用支持发布为：

* 直接运行
* 嵌入网站
* 访问 API

工作流应用支持发布为：

* 直接运行
* 批处理
* 访问 API
* 发布为工具

<Info>
如需管理多个版本的 聊天流/工作流，请参阅 [版本管理](https://docs.dify.ai/zh-hans/guides/management/version-control)。
</Info>
