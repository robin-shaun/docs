天智 是基于大语言模型的 AI 应用开发平台，初次使用时你需要先在天智的 **设置 -- 模型供应商** 页面内添加并配置所需要的模型。

![设置-模型供应商](https://assets-docs.dify.ai/dify-enterprise-mintlify/zh_CN/guides/model-configuration/e83d92f43b9cd8eca719b3c27ca63418.png)

### 模型类型

在天智中，我们按模型的使用场景将模型分为以下 4 类：

1. **大语言模型**。 在创建的应用中，用的是该类型的模型。智聊、对话名称生成、下一步问题建议用的也是推理模型。

   > 已支持的大语言模型：Deepseek R1、Qwen2.5
2. **Embedding 模型**。在知识库中，将分段过的文档做 Embedding 用的是该类型的模型。在使用了知识库的应用中，将用户的提问做 Embedding 处理也是用的该类型的模型。

   > 已支持的 Embedding 模型：m3e large
3. [**Rerank 模型**](/zh-hans/advanced/retrieval-augment/rerank)。**Rerank 模型用于增强检索能力，改善 LLM 的搜索结果。**

   > 已支持的 Rerank 模型：bge ranker
4. **语音转文字模型**。在对话型应用中，将语音转文字用的是该类型的模型。

   > 已支持的语音转文字模型：暂无

根据技术变化和用户需求，我们将陆续支持更多大模型

### 接入模型设置

在**_模型插件_**菜单下选择“安装插件”，选择本地插件“OpenAI_API_compatible.difypkg”。

 在**_模型供应商 _**中选择OpenAI_API_compatible，点击“添加模型”。

![](https://assets-docs.dify.ai/dify-enterprise-mintlify/zh_CN/guides/model-configuration/a8d65b27c806b0a6fabe88099023d116.png)

目前已支持的模型包括：

| 模型类型           | 模型名称                            | API Key | API endpoint URL            |
| -------------- | ------------------------------- | ------- | --------------------------- |
| LLM            | qwen2_5-32b-instruct-gptq-int8, | tianshu | http://10.17.64.128:8105/v1 |
| LLM            | deepseek-r1                     | tianshu | http://10.17.64.190:8105/v1 |
| Text embedding | m3e-large                       | 无       | http://10.17.64.128:7106/v1 |
